Welcome to Template pages

Sample pages and routes to call


  • /cookie - Dump request headers and set a cookie (text reply)
  • /headerfooter - Serve a template page with hardcoded support for header and footer (page reply)
  • /health - Health Check (fixed json reply)
  • /hello - Hello (dynamic text reply)
  • /hj - Hello (fixed json reply)
  • /mystatus - Sample error code (fixed json reply)
  • /not/existent - Call a not existing route (HTTP error 404 expected)
  • /ready - Readiness Check (fixed json reply, but after some seconds, to simulate a real dependency check)
  • /time - Current timestamp (dynamic json reply)
  • /user/1001 - Return some user info, with user id in the path (fixed json reply)
